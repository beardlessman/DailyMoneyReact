# DailyMoney React App

React-версия приложения для отслеживания ежедневных расходов.

## Особенности

- ✅ Работает полностью офлайн (без интернета)
- ✅ Хранение данных в localStorage
- ✅ Расчет дневного бюджета на основе месячного
- ✅ Экспорт лога транзакций
- ✅ Красивый и современный UI

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Функциональность

- **Добавление транзакций**: Ввод суммы и выбор категории
- **Просмотр лога**: Отображение всех транзакций по дням
- **Расчет бюджета**: Автоматический расчет дневного бюджета на основе месячного
- **Экспорт**: Экспорт лога в текстовый файл
- **Настройки**: Изменение месячного бюджета

## Технологии

- React 18
- TypeScript
- Tailwind CSS
- Vite
